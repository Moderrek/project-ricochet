# Menus (Zarządzanie stanem ekranu)
Sceny odpowiadające za nawigację poza właściwą rozgrywką (Core Gameplay Loop).

## Struktura
- `main_menu.tscn`: Ekran startowy.
- `end_screen.tscn`: Ekran podsumowania. Pokazuje się dzięki `GameManager` w przypadku upływu czasu lub ukończenia wszystkich poziomów.
