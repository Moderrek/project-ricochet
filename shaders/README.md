# Shaders (Programowanie GPU)
Niestandardowe materiały wizualne modyfikujące renderowanie pikseli.

## Struktura
- `blueprint.gdshader`: Generator tła wyglądający jak papier milimetrowy.
- `vignette.gdshader`: Efekt winety skupiający wzrok na centrum ekranu.
- `background_blur.gdshader`: Efekt rozmycia. Wykorzystywany w menu pauzy lub ekranach nakładkowych.
